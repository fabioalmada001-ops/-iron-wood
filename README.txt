IRON WOOD - SITIO WEB INICIAL

Archivos:
- index.html: estructura y contenido.
- styles.css: diseño responsive.
- script.js: menú, filtros, personalizador, modal y demostración de IA.
- assets/: imágenes de referencia.

Cómo abrir:
1. Descomprimí el archivo.
2. Abrí index.html con Chrome, Edge o Firefox.

Para publicar gratis:
- GitHub Pages
- Netlify
- Vercel

Antes de publicar:
- Cambiar el número de WhatsApp en index.html.
- Agregar precios y datos reales.
- Reemplazar imágenes de referencia por fotos propias cuando estén disponibles.
- Conectar el formulario con correo, WhatsApp o una base de datos.
- Integrar una API real para la generación con IA.
